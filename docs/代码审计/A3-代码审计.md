| 1 | `GZ:` 压缩前缀格式 | **A. 保持 `GZ:` 前缀** — 简单高效，JSON 以 `{`/`[` 开头零误判。B. JSON 信封 `{"c":true,"d":"..."}` — 增加解析开销。C. 独立列 — 需 ALTER TABLE。D.
